# Phần mềm quản lý khách sạn
## 1. Tổng quan:
- Ngôn ngữ: Java Swing
- Loại CSDL: Relational Database
- Đối tượng sử dung: Người quản lý, Lễ tân
- Các chức năng chính: Quản lý các nghiệp vụ của khách sạn, hỗ trợ đặt phòng, theo dõi doanh thu theo tháng
